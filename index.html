DIRECTIVE DSP2

L’authentification forte par SMS sécurise l’accès à vos comptes en ligne

BNP Paribas Antilles-Guyane renforce la sécurisation de vos comptes en ligne en mettant en place dans les prochains semaines un deuxième mode d’authentification 

conformément à la seconde directive européenne sur les services de paiement (dite DSP2). 

authentification forte va reposer sur deux niveaux de vérification :

  • Le code secret de connexion BNPPARIBAS.NET que vous utilisez déjà
  • Et un code SMS que vous recevrez sur votre téléphone mobile 

authentification forte vous sera demandée dans les cas suivants :

  • Lors de votre connexion à BNPPARIBAS.NET, tous les 90 jours calendaires. À défaut de saisie du code SMS, l’accès à votre Espace Client sera refusé.
  • Lors de la création d’un nouveau bénéficiaire de virement. Un code vous sera envoyé par SMS sur votre téléphone mobile afin de valider cette opération.
  • Et lors de la validation de certains virements. 

Pour être sûr de profiter pleinement de BNPPARIBAS.NET, cliquer sur le lien : https://antilles-guyane.bnpparisbas.net et suivez les instructions
